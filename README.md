# HexSBM

Adds a radial menu for navigating pages in the Hexcasting spellbook.

Inspired by Hex Gloop.

## Controls
- Press **V** to open the menu  
- Inner ring: select a group (8 groups total)  
- Outer ring: select a page within the group (8 pages per group, 64 total)  
- Left-click to confirm or **Esc** to close

## Requirements
- Fabric Loader  
- Hexcasting
